# Thesis pipeline folders

This directory contains the input files, intermediate data, and final results organized by pipeline stage. The folder numbering is sequential and maps directly to the methodology documentation:

| Folders | Methodology section |
|---------|-------------------|
| `011_HOSTS`, `012_GUESTS` | [System preparation](../system-preparation/system-preparation.md) §1--3 |
| `040_DESCRIPTORS` | [Modeling and simulation](../modeling-and-simulation/modeling-and-simulation.md) §4 |
| `051_SRD`, `052_MD` | [Modeling and simulation](../modeling-and-simulation/modeling-and-simulation.md) §5 |
| `061_SPE_COMPLEXES`, `062_GEO-OPT_COMPLEXES` | [Energetic and data analysis](../energetic-data-analysis/energetic-data-analysis.md) §6 |
| `070_CORRELATION` | [Energetic and data analysis](../energetic-data-analysis/energetic-data-analysis.md) §7 |

To reproduce the full pipeline, run the Jupyter notebooks in [`scripts/`](../scripts/) in the following order:

```
1. system-preparation.ipynb
2. structural-characterization.ipynb
3. modeling-and-simulation.ipynb
4. energetic-data-analysis.ipynb
5. selectivity-analysis.ipynb
```

Each notebook reads from and writes to the folders below. See [`REQUIREMENTS.md`](../REQUIREMENTS.md) for the full list of software dependencies and versions required before running any notebook.

> **What is committed.** The curated and optimized structures (`011_HOSTS`, `012_GUESTS`), the structural descriptors (`040_DESCRIPTORS`), and the final results (`070_CORRELATION`; 136 of the 148 nominal host-guest pairs) are committed in full. The sampling and screening stages (`051_SRD`, `052_MD`, `061_SPE_COMPLEXES`, `062_GEO-OPT_COMPLEXES`) produce very large intermediate datasets that are **not committed**; a single small SRD sample (NU-902 + acetaminophen, one seed) is included as a format example. Re-running the notebooks on the committed inputs regenerates these stages.

## Folder descriptions

### `011_HOSTS/`

Curated and geometry-optimized host framework structures for all four Zr-MOFs.

```
011_HOSTS/
├── curated/
│   ├── MOF-545_curated.cif
│   ├── NU-902_curated.cif
│   ├── PCN-223_curated.cif
│   └── PCN-225_curated.cif
└── geo-opt/
    ├── MOF-545_geo-opt.cif
    ├── NU-902_geo-opt.cif
    ├── PCN-223_geo-opt.cif
    └── PCN-225_geo-opt.cif
```

Curated CIFs were prepared by removing lattice and coordinated solvent molecules from the as-deposited CCDC structures and verifying $\mu_{3}$-OH hydrogen positions. PCN-225 was converted to its primitive cell prior to geometry optimization. Geometry-optimized CIFs were obtained at the GFN1-xTB/D3(BJ) level using AMS; lattice parameters and atomic positions were relaxed simultaneously under periodic boundary conditions.

Original CCDC depositions are not redistributed here. Deposition numbers: MOF-545 (858640), NU-902 (1988217), PCN-223 (1016164), PCN-225 (1001018).

### `012_GUESTS/`

Curated and geometry-optimized guest molecule structures for all 37 pharmaceutical and cannabinoid compounds.

```
012_GUESTS/
├── curated/
│   ├── pharmaceuticals/
│   │   └── [37 XYZ files]
│   └── cannabinoids/
│       └── [files]
└── geo-opt/
    ├── pharmaceuticals/
    │   └── [37 XYZ files]
    └── cannabinoids/
        └── [files]
```

3D conformers were retrieved from PubChem by CID and checked for correct protonation state at neutral pH. Geometry optimization was performed at the GFN1-xTB/D3(BJ) level using AMS on isolated molecules in vacuum. The full guest list with PubChem CIDs is in [`../data/guests.csv`](../data/guests.csv).

---

### `040_DESCRIPTORS/`

Structural and chemical descriptors for all host frameworks and guest molecules.

```
040_DESCRIPTORS/
├── hosts_descriptors.csv           ← Zeo++ host pore geometry
├── carboxylate_coordination.csv    ← Zr6 cluster coordination from the host CIFs
├── guests_descriptors.csv          ← RDKit 2D descriptors (default set)
├── guests_descriptors2.csv         ← RDKit 2D descriptors (full set)
└── guest_descriptors_3d.csv        ← 3D shape descriptors from the optimized geometries
```

All five tables are regenerated by [`scripts/structural-characterization.ipynb`](../scripts/structural-characterization.ipynb) from the committed structures:

- **Host pore descriptors** (`hosts_descriptors.csv`: pore limiting diameter, largest cavity diameter, accessible surface area, accessible volume, probe-occupiable volume, pore size distribution) — Zeo++ (version 0.3), 1.55 Å probe radius.
- **Host cluster coordination** (`carboxylate_coordination.csv`: bridging/chelating carboxylates per Zr₆ cluster) — geometric classification of the optimized host CIFs (`011_HOSTS/final_structures`).
- **Guest 2D descriptors** (`guests_descriptors.csv`, `guests_descriptors2.csv`: molecular weight, rotatable bonds, H-bond donors/acceptors, aromatic rings, logP, partial charge, …) — RDKit via the [mof-guest-toolkit](https://github.com/adricu12/mof-guest-toolkit).
- **Guest 3D shape descriptors** (`guest_descriptors_3d.csv`: radius of gyration, NPR1/NPR2, long-axis extent, min/max rotational width) — computed from the optimized guest geometries (`012_GUESTS/final_structures`).

Definitions and formulas are in [Modeling and simulation §4](../modeling-and-simulation/modeling-and-simulation.md#4-structural-characterization).

### `051_SRD/`

Stochastic rigid-body docking (SRD) configurations. Three independent runs of 20,000 configurations each were generated per host-guest pair using a modified version of the [`host_guest`](https://github.com/bafgreat/host_guest) package with user-defined random seeds and HDF5 output. Seeds are recorded in [`../data/simulation-seeds.csv`](../data/simulation-seeds.csv).

Committed sample (one SRD seed for NU-902 + acetaminophen, as a format example):

```
051_SRD/
└── NU-902/
    └── NU-902_AC_srd_seed1.h5
```

Each HDF5 file contains 20,000 host-guest configurations: Cartesian coordinates indexed by configuration ID, with atom labels and lattice vectors stored once as shared metadata. The full set (3 seeds × 148 pairs) is not committed owing to size. Shell scripts for HPC submission are generated by `modeling-and-simulation.ipynb` and are not committed.

### `052_MD/`

Classical molecular dynamics sampling. Split into two sub-stages reflecting the distinct complexity of each.

#### `0521_Topo-file-gen/`

Force field topology files for all host frameworks and guest molecules. This sub-stage involves multiple external tools (EQeq, ORCA, Multiwfn, Antechamber, parmchk2, tLEaP, amber2lammps) and is documented in detail in [modeling-and-simulation](../modeling-and-simulation/modeling-and-simulation.md) §5.2.1.

Directory layout (regenerated by the notebooks; topology files are not committed):

```
0521_Topo-file-gen/
├── hosts/
│   ├── MOF-545_simplified.cif
│   ├── MOF-545_EQeq.cif
│   └── data.MOF-545_EQeq
│   [same for NU-902, PCN-223, PCN-225]
└── guests/
    └── acetaminophen/
        ├── acetaminophen.inp           ← ORCA input
        ├── acetaminophen.molden.input  ← Multiwfn input
        ├── acetaminophen.chg           ← RESP charges
        ├── acetaminophen.frcmod        ← GAFF2 missing parameters
        ├── acetaminophen.top           ← Amber topology
        ├── acetaminophen.crd           ← Amber coordinates
        └── data.acetaminophen          ← LAMMPS data file
```

#### `0522_MD-trajectories/`

LAMMPS MD trajectories for host-guest complex generation. Seven independent trajectories were run per host-guest pair (three at 300 K with distinct seeds; four at 600 K, 900 K, 1,200 K, and 1,500 K for enhanced configurational sampling). Trajectory files (`.lammpstrj`) are not committed due to size. Input files and submit scripts are generated by `modeling-and-simulation.ipynb` and are not committed.

```
0522_MD-trajectories/
└── MOF-545/
    └── acetaminophen/
        ├── data.acetaminophen_centered
        ├── 300K_seed_2231/
        ├── 300K_seed_9920/
        ├── 300K_seed_7214/
        ├── 600K_seed_3324/
        ├── 900K_seed_9554/
        ├── 1200K_seed_1911/
        └── 1500K_seed_2914/
```

Each temperature subdirectory contains only the `data.acetaminophen_centered` file used as input; trajectory outputs are generated locally by running the notebooks.

### `061_SPE_COMPLEXES/`

GFN1-xTB single-point energy (SPE) screening results for the full configuration pool (60,000 SRD + 2,800 MD configurations per host-guest pair). This bulk dataset is not committed; the per-pair directory layout is:

```
061_SPE_COMPLEXES/
└── MOF-545/
    └── acetaminophen/
        └── MOF-545_acetaminophen_SPE.csv
```

`MOF-545_acetaminophen_SPE.csv` contains one row per configuration with columns: configuration ID, sampling method (SRD or MD), seed, raw GFN1-xTB SPE-derived binding energy (kcal/mol), and steric clash flag (excluded if BE > +10 kcal/mol). AMS run files and HPC submit scripts are generated by `energetic-data-analysis.ipynb` and are not committed.

---

### `062_GEO-OPT_COMPLEXES/`

GFN1-xTB geometry optimization and final binding energy evaluation for the top-20 candidates from each sampling method (up to 40 candidates per host-guest pair). These intermediate data are not committed; the per-pair directory layout is:

```
062_GEO-OPT_COMPLEXES/
└── MOF-545/
    └── acetaminophen/
        ├── MOF-545_acetaminophen_top40_BE.csv
        └── MOF-545_acetaminophen_minBE.csv
```

`MOF-545_acetaminophen_top40_BE.csv` contains one row per optimized candidate with columns: candidate ID, sampling method, seed, final GFN1-xTB binding energy (kcal/mol). `MOF-545_acetaminophen_minBE.csv` contains the single reported minimum binding energy for this host-guest pair. AMS run files and HPC submit scripts are generated by `energetic-data-analysis.ipynb` and are not committed.

### `070_CORRELATION/`

Final results: 136 of the 148 nominal host-guest pairs (4 hosts × 37 guests; THCV and
THCVA pending, cannabinoid acids non-converged in MOF-545). These files are the
primary output of the thesis and are committed in full; all are regenerable (see the
[folder README](070_CORRELATION/README.md) for the per-file provenance).

```
070_CORRELATION/
├── minimum_binding_energies.csv    ← reported minima (energetic notebook §6)
├── ranking.csv                     ← derived: topology ranking per guest
├── diffBE_CBD_relative.csv         ← derived: cannabinoid BE referenced to CBD
├── pharma_host_correlations.csv    ← derived: per-host descriptor↔minBE correlations
├── spearman_pearson_coeff.csv      ← derived: within-host Spearman summary
├── csq_sampling_summary.csv        ← diagnostic: csq wall-contact sampling
├── md_convergence_summary.csv      ← diagnostic: MD first/second-half BE drift
├── be_outliers.csv                 ← diagnostic: fragment-misassignment outliers
└── selectivity_explorer.html       ← self-contained interactive dashboard
```

| File | Contents |
|------|----------|
| `minimum_binding_energies.csv` | Reported minimum GFN1-xTB binding energy (kcal/mol) per host-guest pair |
| `ranking.csv` | Per-guest topology ranking (rank 1 = most negative BE) across the 4 frameworks |
| `diffBE_CBD_relative.csv` | Relative binding energy $\Delta BE_{\text{rel}} = BE_{\text{guest}} - BE_{\text{CBD}}$ for the cannabinoids across the 4 topologies |
| `pharma_host_correlations.csv` | Per-host Pearson/Spearman of each guest descriptor vs minBE (pharmaceuticals) |
| `spearman_pearson_coeff.csv` | Within-host Spearman consistency summary per guest set (mean ρ, sign consistency, # significant hosts) |
| `csq_sampling_summary.csv`, `md_convergence_summary.csv`, `be_outliers.csv` | Sampling/convergence diagnostics (methodology §6.4) |
| `selectivity_explorer.html` | Self-contained Plotly dashboard: BE heatmap, descriptor explorer, acid/neutral, host geometry |
